# Convert tidyGWAS to file with RSID and P for clumping

Convert tidyGWAS to file with RSID and P for clumping

## Usage

``` r
to_clumping(path)
```

## Arguments

- path:

  filepath to tidyGWAS folder

## Value

writes out a tsv.gz file

## Examples

``` r
if (FALSE) { # \dontrun{
to_clumping("/path/to/tidyGWAS")
} # }
```
