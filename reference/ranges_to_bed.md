# Convert the output of plink clumping ranges to a flat file

Convert the output of plink clumping ranges to a flat file

## Usage

``` r
ranges_to_bed(clump_dir)
```

## Arguments

- clump_dir:

  directory where plink clumping output is stored

## Examples

``` r
if (FALSE) { # \dontrun{
ranges_to_bed("/path/to/clumping/dir")
} # }
```
