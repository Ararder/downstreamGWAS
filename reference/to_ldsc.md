# Export tidyGWAS format to LDSC format

Export tidyGWAS format to LDSC format

## Usage

``` r
to_ldsc(parent_folder, use_effective_n = TRUE)
```

## Arguments

- parent_folder:

  filepath to tidyGWAS folder

- use_effective_n:

  Should an attempt be made to calculate effective N and use that as N?

## Examples

``` r
if (FALSE) { # \dontrun{
to_ldsc("my_sumstats/tidygwas/height2022")
} # }
```
