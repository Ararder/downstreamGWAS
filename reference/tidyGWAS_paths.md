# Create the folder structure and filepaths for downstreamGWAS directory

Create the folder structure and filepaths for downstreamGWAS directory

## Usage

``` r
tidyGWAS_paths(dir)
```

## Arguments

- dir:

  filepath to folder

## Value

a list of filepaths

## Examples

``` r
if (FALSE) { # \dontrun{
 tidyGWAS_paths("gwas/height2022")

```
